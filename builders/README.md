# The Builder Wall

Every OneMillion graduate gets a permanent, public, sequential **Builder #N**. The wall is here.

**Current Builder count:** see `_data.json`

---

## How To Read This Wall

Each entry is a real builder who:
- Completed all 18 days of OneMillion
- Passed all 18 daily verifications
- Shipped a live product deployed at their URL
- Submitted a Demo Day Loom

Click any builder's number to see their full profile (deployed URL, 18-day commit history, demo Loom, what they built).

## How To Get Your Number

1. Complete all 18 days of the course
2. Pass all 18 Claude Code verification checks
3. Record your Day 18 demo Loom
4. Submit a **Builder Claim** issue in this repo
5. Sid reviews the claim
6. Accepted claim receives the next sequential Builder #N

Builder Wall profile pages and LinkedIn badge images are planned automation. Until then, `_data.json` is the source of truth.

---

## Privacy

What's public on your profile:
- Builder number
- Display name (your choice — real name, alias, anything non-offensive)
- Your product URL
- Your demo Loom URL
- Your GitHub repo URL (if you marked it public)
- Cohort number (if any)

What's NOT public:
- Email
- Real name (unless you put it in display name)
- API key usage / costs
- Verification reports (those stay in your repo)

You can request removal at any time. Once removed, the number is retired (we don't reassign it).
