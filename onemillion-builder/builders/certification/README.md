# OneMillion Builder Certificate

**Earn your certificate by shipping a real product.**

## Requirements

To earn the OneMillion Builder Certificate, you must complete all 18 days and provide:

1. **Live URL** — your product is deployed and accessible
2. **GitHub repo URL** — source code is public
3. **Build journal** — 18 entries (one paragraph per day)
4. **The 5 artifacts** — all present in your `.onemillion/` directory:
   - `prd.md`
   - `refined-prd.md`
   - `architecture.md`
   - `test-results.md`
   - `security-audit.md`

## How to Claim

Run the certification script in your project directory:

```bash
claude
```

Then type:
```
/sell Generate my completion certificate
```

The SELL agent will verify your artifacts, generate a PDF certificate, and post a verification record.

## What the Certificate Proves

- You shipped a production-grade web application from zero
- You wrote no boilerplate — everything was generated through structured agents
- Your product passed security audit (no Critical/High findings)
- Your product has E2E tests and meets WCAG accessibility standards
- You understand each phase of the build pipeline well enough to direct it

## Certificate Levels

| Level | Requirement |
|-------|-------------|
| **Builder** | All 18 days completed, product live |
| **Builder Pro** | Builder + 10 real users in the first week |
| **Builder Expert** | Builder Pro + product generates revenue |

Submit evidence for Pro and Expert levels via the community forum.
