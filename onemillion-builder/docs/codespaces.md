# Codespaces Fallback

Use this only if your laptop cannot run a local editor or development server.

The primary course path is harness-neutral:

```text
favorite coding harness + AGENTS.md + course manifest
```

Codespaces can still help when you need a browser-based development machine.

---

## When To Use This

Use Codespaces if:

- you are on a locked-down work laptop
- you are on a Chromebook
- Node/Git setup is blocked locally
- your local app will not run and you need a clean environment

---

## Basic Flow

1. Create a GitHub account.
2. Open [github.com/codespaces](https://github.com/codespaces).
3. Create a blank Codespace or open your app repo in Codespaces.
4. Use the Codespaces terminal for app commands.
5. Keep your coding harness pointed at the course repo so it can read `AGENTS.md` and guide you.

---

## Important Limitation

Codespaces is a development environment fallback, not the course brain.

The course brain is still:

```text
AGENTS.md
onemillion-builder/course-manifest.json
onemillion-builder/agents/
```
