# Gemini

Open the cloned repo in Gemini CLI or a Gemini-based coding harness.

Run the installer first:

```bash
./onemillion-builder/install-agents.sh
```

Start with:

```text
I am starting the OneMillion course. Read AGENTS.md, onemillion-builder/course-manifest.json, onemillion-builder/docs/teaching-protocol.md, onemillion-builder/course/single.md, and .gemini/GEMINI.md, then start Day 0. First verify fork, clone, origin, and upstream. Properly greet me, explain the course, explain the OneMillion development pipeline, provide copy-ready actions, and define what done means.
```

If your Gemini setup supports extensions, treat `onemillion-builder/agents/` as the portable source of agent instructions.
