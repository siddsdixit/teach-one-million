## What To Build Next

After Day 18, build another product. Then another. Each one teaches new patterns:

| Project type | What you learn |
|--------------|---------------|
| **CRUD SaaS** (the course) | Auth, database, deployment, real users |
| **AI Agent** (next) | Tool use, scheduled jobs, memory, autonomy |
| **Chrome extension + AI** | Browser APIs, content scripts, smaller scope |
| **Webhook automation** (Zapier-like) | API integration, event-driven design |
| **Voice or video AI** | Multimodal models, streaming media |
| **Internal tool for a company** | Real users, real feedback, possibly real revenue |

Pick the one closest to a problem you want solved.
