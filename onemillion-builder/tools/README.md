# Tools — Pick Your AI Builder

The OneMillion course is **tool-agnostic**. The IP is the framework — 5 pillars + 18-day journey + verification + Builder #N. The agents are reference implementations. You can complete the entire course in whichever AI builder you prefer.

There are **three tiers of fidelity**:

## Tier 1: First Class — Claude Code

The cleanest fit. The course's agents are Claude Code agents (subagents, model routing, slash commands). If you have no preference, use this.

→ **[Tier 1: Claude Code setup](./tier-1-claude-code/setup.md)**

## Tier 2: Persona-Switching — Cursor / Antigravity / Windsurf

Excellent IDEs with their own AI agents (Composer, Cascade, etc.). The course works in all of them. You use a single mega-rule that contains all 9 OneMillion agent personas — your main AI adopts the right persona based on context. Loses true subagent isolation but works clean enough.

→ **[Tier 2: Cursor](./tier-2-cursor/setup.md)**
→ **[Tier 2: Antigravity](./tier-2-antigravity/setup.md)** (limited support — research-grade)
→ **[Tier 2: Windsurf](./tier-2-windsurf/setup.md)**

## Tier 3: Universal Prompts — Any AI Chat

Works in Claude.ai web, ChatGPT, Gemini, Mistral, anything that accepts paste-able prompts. Lowest fidelity (no file system access, no autonomous execution), but maximum accessibility. Best for builders who can't install anything locally.

→ **[Tier 3: Universal prompts](./tier-3-universal-prompts/README.md)**

---

## How To Pick

| You... | Use |
|--------|-----|
| Want the cleanest experience | Tier 1 (Claude Code) |
| Already pay for Cursor / use it daily | Tier 2 (Cursor) |
| Are a Google AI early adopter | Tier 2 (Antigravity) |
| Want to keep your existing AI tool | Tier 2 (Windsurf) or Tier 3 |
| Can't install software (corporate laptop, etc.) | Tier 3 + Codespaces fallback |

---

## Important: One AI Tool, Not Many

Use ONE AI tool consistently through the course. Don't:
- ❌ Use Cursor's Composer for Day 4, then switch to Claude Code for Day 5
- ❌ Try to run OneMillion agents in two tools simultaneously

The course assumes consistency. Pick one. Stick with it. You can experiment with others *after* you graduate.

---

## What If The Tool Changes Mid-Course?

If your chosen AI tool releases a major update during your 18 days (new pricing, new features, etc.), don't switch. Finish in your current tool. Re-evaluate after Day 18.

The framework outlasts any specific tool. Your skill transfers.
