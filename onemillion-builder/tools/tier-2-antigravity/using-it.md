# Tier 2: Using Antigravity Daily

**Status: Research-grade.** Patterns updating as Antigravity matures.

## Recommended Path

Use Antigravity as your editor + Claude Code in its integrated terminal. This gives you the best of both:
- Antigravity's editor + native AI features when you want them
- Claude Code (Tier 1) for the OneMillion agentic SDLC flow

Follow [Tier 1: Using Claude Code Daily](../tier-1-claude-code/using-it.md) — substitute "Antigravity terminal" for "VS Code terminal."

## If You Want To Use Antigravity's Native Agent

The patterns are similar to Cursor (Tier 2):

1. Reference the day number in your messages
2. Paste day's build.md to start
3. End day with the verification prompt

The mega-rule from Cursor (`tools/tier-2-cursor/onemillion-rule.mdc`) can be adapted to Antigravity's rule syntax. See Antigravity's docs for the exact format.

## Known Limitations

- Smaller community = harder to Google when stuck
- Less stable rule format (may change in updates)
- Persona switching may be less reliable than Cursor

## When To Reconsider

If Antigravity matures over the next 6 months and the OneMillion community produces stable patterns, this tier upgrades. For now: editor + Claude Code is the safe path.
