# Tier 2: Windsurf Setup

Windsurf is Codeium's AI-first IDE, similar to Cursor. The OneMillion course works in it.

## Install

1. Download Windsurf at [codeium.com/windsurf](https://codeium.com/windsurf)
2. Sign in (free tier available)

## Use With OneMillion

### Option A: Claude Code In Windsurf's Terminal (RECOMMENDED)

Same as Tier 1 — open Windsurf, open its integrated terminal, install Claude Code, proceed.

### Option B: Windsurf's Cascade Agent

Cascade is Windsurf's built-in AI agent (similar to Cursor's Composer).

1. Copy the OneMillion mega-rule from `tools/tier-2-cursor/onemillion-rule.mdc` to Windsurf's rule location (check Windsurf docs)
2. Open Cascade
3. Tell it which day you're on, paste build.md instructions
4. Iterate

## Tradeoffs

| | Cursor | Windsurf |
|--|--|--|
| Maturity | More mature | Newer but stable |
| Pricing | Pro: $20/mo | Free tier generous |
| Community | Larger | Smaller |
| OneMillion patterns | Documented (tier-2-cursor) | Adaptable |

If you're already using Windsurf: stick with it. The course works fine.

If you're new: try Cursor first (more docs, larger community).
