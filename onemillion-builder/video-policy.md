# Video Policy

The written course must be complete without video. Videos increase completion and confidence, but a missing video must never block the learner.

## Public Page Rules

Use this when the video exists:

```md
## Watch First

[Loom — Day X walkthrough](https://www.loom.com/share/...)
```

Use this when the video does not exist yet:

```md
Video walkthrough: coming soon. The written guide is complete.
```

Do not use:

- placeholder status text
- `placeholder`
- fake `#` links
- internal sprint names
- Video embed labels without a real URL

## Update Order

1. Add the Loom URL to [loom-inventory.md](loom-inventory.md).
2. Update the day's `loom.md`.
3. Update the day's `learn.md` if it has a Watch First section.
4. Run the markdown link check.
