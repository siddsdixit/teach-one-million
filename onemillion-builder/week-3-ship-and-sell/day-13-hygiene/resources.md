# Day 13 — Resources

## Security checklists
- **[OWASP Top 10](https://owasp.org/www-project-top-ten/)** — canonical web app vulnerabilities
- **[OWASP LLM Top 10](https://owasp.org/www-project-top-10-for-large-language-model-applications/)** — AI-specific risks

## Tools
- **[gitleaks](https://github.com/gitleaks/gitleaks)** — secret scanner you used today
- **[git-filter-repo](https://github.com/newren/git-filter-repo)** — remove secrets from git history
- **[trufflehog](https://github.com/trufflesecurity/trufflehog)** — alternative secret scanner with more rules
- **[Vercel KV](https://vercel.com/docs/storage/vercel-kv)** — Redis-compatible key-value store for rate limiting

## Patterns
- **[Vercel Security Headers](https://vercel.com/docs/security/headers)** — defense in depth
- **[Anthropic Prompt Injection](https://docs.anthropic.com/en/docs/build-with-claude/prompt-injection)** — AI-specific risks
- **[Cloudflare WAF](https://www.cloudflare.com/learning/insights-network-services/what-is-a-web-application-firewall-waf/)** — when you scale beyond Vercel's defaults
